# EIC (Education Information Center)

Мобильное приложение для Android, предназначенное для студентов. Объединяет функциональность для работы с учебными данными и сообщениями в одном интерфейсе.

**Текущий статус:** Демонстрационная версия (Demo) 

##  Основные возможности 

*   **Интеграция с Eljur:** Получение и отображение расписания, заданий и учебных материалов.
*   **Интеграция с Telegram:** Просмотр и управление сообщениями через официальную библиотеку TDLib.
*   **Локальное хранение:** Кэширование данных на устройстве для быстрого доступа и работы офлайн.
*   **Удобный интерфейс:** Построен с использованием современных компонентов Android.

##  Стек технологий

*   **Языки:** Java (основной), Kotlin
*   **Минимальная SDK:** Android 8.0 (API level 24)
*   **Инструменты сборки:** Gradle (Kotlin DSL)
*   **Архитектура:** Приложение находится в стадии разработки архитектурных компонентов

##  Запуск и сборка

1.  **Клонируйте репозиторий**
    ```bash
    git clone https://github.com/vernuch/EljurTelegramDemo.git
    cd EljurTelegramDemo
    ```

2.  **Откройте проект в Android Studio**
    *   Для корректной работы рекомендуется использовать **Android Studio Flamingo (2025.1.3) или новее**, так как проект использует оболочку Gradle 8.5.
    *   Дождитесь завершения индексации и синхронизации проекта (Gradle Sync).

3.  **Сборка проекта**
    *   Для сборки отладочной (debug) версии используйте стандартную команду `Build > Make Project` или запустите таск `assembleDebug` из панели Gradle.
    *   Собранный APK-файл для тестов уже доступен в корне репозитория: `app-debug-androidTest.apk`.

## Текущая реализация и заметки для разработки

*   **Кодовая база:** Основная логика приложения в данный момент написана на **Java**.
*   **Управление зависимостями:** Все зависимости объявлены в файле `app/build.gradle.kts`.
*   **Интеграция:** Реализован модуль для интеграции с Telegram при помощи TDLib, модуль интеграции Eljur не было возможности протестировать
*   **Универсальность:** В проект не требует обязательных авторизаций, имеется логика ручного заполнения всех разделов
